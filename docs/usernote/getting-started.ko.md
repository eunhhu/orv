# 시작하기

[← 사용자 문서로 돌아가기](./README.ko.md)

---

이 저장소는 아직 초기 단계입니다. 문서는 제품 방향, 언어 레퍼런스, 구현 명세가 혼합되지 않고 분리되도록 구성되어 있습니다.

## 현재 존재하는 것

- 저장소에는 `orv-cli`, `orv-core`, `orv-macros`를 포함한 `orv`의 핵심 크레이트가 포함되어 있습니다.
- 현재 코드상의 CLI 인터페이스는 최소한의 수준입니다. 구현된 명령 경로는 버전 출력뿐입니다.
- 언어 레퍼런스는 이미 [`syntax/`](./syntax/) 아래에 모듈화되어 있습니다.

## 권장 첫 단계

1. [왜 orv인가](./why-orv.ko.md)를 읽고 프로젝트 목표를 이해합니다.
2. [언어 레퍼런스](./syntax.ko.md)에서 현재 구문 범위를 확인합니다.
3. [설계 명세](../specs/README.md)에서 최적화 및 컴파일러/런타임 제안을 확인합니다.

## 현재 CLI 스냅샷

```bash
cargo run -p orv-cli
cargo run -p orv-cli -- version
```

더 많은 CLI 명령이 구현되면, 이 페이지는 작업 중심의 설정 및 워크플로우 가이드로 확장될 예정입니다.
